## Тематика сайта, название

Игра "Math madness". Идея заключается в реализации игры по пободобию [Flappy Bird](https://ru.wikipedia.org/wiki/Flappy_Bird). Игра представляет собой бесконечный раннер,но с элементами,которые относяться к математике,к примеру: задний фон будет украшен различными формулами, препятствиями будут являться:число пи, число эйлера. В случае проигрыша игроку будет предоставляться выбор: либо начать с самого начала, либо решить алгебраическую задачу и продолжить игру с места проигрыша. Этот этап будет заменять рекламу в аналогичных продуктах.

## Целевая аудитория

Целевая аудитория это любой человек,который хочет 'убить' время и немножко порешать простейшие задачки,которые еще и не являются обязательными, на уровне 9-10 классов.Игра обычный таймкиллер,поэтому найдет отклик у аудитории любого возраста,аудитория отличается от обобщенного пользователя Интернета тем, что их трата времени сопровождается не только развлечением, но и лёгкой интеллектуальной нагрузкой.

## Функциональные возможности

* возможность смены цвета заднего фона;
* настройка сложности(чем выше сложность,тем больше препятствий);
* возможность изменить облик персонажа;
